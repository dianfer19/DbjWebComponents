Loaders

Indicadores de carga minimalistas y performantes.

Ring Loader CSS only
Ver código 
<!-- HTML -->
<div class="ring" role="status" aria-label="Cargando"></div>

/* CSS */
.ring{width:58px;height:58px;border-radius:50%;
  border:5px solid transparent;border-top-color:var(--primary);
  border-left-color:var(--primary-600);animation:spin 1s linear infinite}
@keyframes spin{to{transform:rotate(360deg)}}
Accesible, simple y fácil de tematizar con variables.

Inputs

Campos listos para UX moderna (focus, label flotante, accesibilidad).

Input · Label flotante CSS only
Ver código 
<div class="field">
  <input id="name" type="text" placeholder=" " aria-labelledby="lbl-name">
  <label id="lbl-name" for="name">Tu nombre</label>
</div>

/* CSS base en styles.css (ver .field, label, focus) */
Compatible con validación nativa; usa :placeholder-shown para animar.
